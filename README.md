# ChessGame
A chess game made with Vue.js and Socket.io for YouStock :)

### Used libraries
- VueJs
- Socket.io
- Chess.js (for game validation)
- Chessground.js
- VueCookies
- canvas-confetti

## Audio sources
All audio are downloaded on [Epidemic Sounds](https://www.epidemicsound.com)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
